# Nexora E-Commerce Platform

A full-stack e-commerce web application built using React, Spring Boot, MySQL, JWT Authentication, and Gmail SMTP integration.

## Tech Stack

### Frontend
- React.js
- JavaScript
- HTML5
- CSS3

### Backend
- Spring Boot
- Spring Security
- JWT Authentication
- JavaMail Sender

### Database
- MySQL

## Features

- User Registration & Login
- JWT Authentication
- Product Management
- Shopping Cart
- Order Placement
- Order Confirmation Email
- Secure REST APIs
- MySQL Database Integration

## Email Integration

Order confirmation emails are automatically sent to customers using Gmail SMTP when an order is placed successfully.

## Project Structure

Frontend:
- React.js

Backend:
- Spring Boot
- MySQL
- JWT Authentication
- Gmail SMTP
